Release Notes

2016/03/11

Data updates up to Feb 2016, new web address (for real), two new methodology topics.

The following changes have been delivered in this release:

  1. Update: Data download files updated up to February 2016 (when applicable).
  2. New: The URL for the PRU Website is now ansperformance.eu.
  3. New: Horizontal en-route flight efficiency key performance indicators (KEP/KEA) included in data.
  4. New: added two topics in Methodology section.
  5. Update: additions and enhancements in Definitions.

Furthermore, we would like to take the opportunity to inform you that the final draft of the Performance Review Report 2015 is now available for stakeholder comments and feedback. Deadline for receipt of comments is 16:00 on Friday 25 March 2016.
